# 🎓 UniSphere – University Management System

UniSphere is a web-based University Management System developed using Django and Python.  
It provides a simple and efficient platform to manage university records with secure authentication and a modern user interface.

---

## 🚀 Features

- 🔐 User Authentication (Login / Logout)
- ➕ Add University Records
- 📋 View All Universities
- ✏️ Update University Details
- ❌ Delete University Records
- 📱 Responsive UI using Bootstrap
- ✅ Data validation and unique constraints

## ⚙️ Installation & Setup

1. Clone the repository

git clone https://github.com/your-username/unisphere.git

cd unisphere
2. Create virtual environment

python -m venv venv
venv\Scripts\activate
3. Install dependencies

pip install django

4. Run migrations

python manage.py makemigrations
python manage.py migrate

5. Create superuser

python manage.py createsuperuser


6. Run the server

python manage.py runserver


7. Open in browser

http://127.0.0.1:8000/
